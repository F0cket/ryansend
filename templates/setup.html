💡 Manual Certificate Configuration:

If you prefer not to use Let's Encrypt, you can manually place certificate files and configure them in config.yaml:

  • Set cert_path: /path/to/cert.pem
  • Set cert_key_path: /path/to/key.pem
  • Set use_letsencrypt_cert: false

You can also use environment variables: RYANSEND_CERT_PATH and RYANSEND_CERT_KEY_PATH
